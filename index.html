🌿 Skincare Routine Style

  • 🧴 Cleanser: Massage for 60 seconds, rinse with lukewarm water.
  • 🧴 Toner: Pat gently using a cotton pad to balance skin pH.
  • 🧴 Serum: Apply 2–3 drops and press lightly for absorption.
  • 🧴 Moisturizer: Massage evenly to hydrate and lock moisture.
  • 🌞 Sunscreen: Apply SPF 30+ 15 mins before stepping out.
  • 🌙 Night Cream: Apply before bed for overnight repair.
  • 💄 Lipstick: Apply from center outward, blot gently for matte look.
  • 👁️ Eyeliner: Glide close to lash line for soft definition.
  • 🧴 Foundation: Blend evenly across face and neck.
  • 🧴 Highlighter: Dab on cheekbones, nose bridge, and cupid’s bow.
  • 💆‍♀️ Face Mask: Use twice weekly for hydration & glow.
  • 🧴 Exfoliator: Use once or twice weekly to remove dead skin.

🕒 When to Use

  • 🌅 Morning: Cleanser → Toner → Serum → Moisturizer → Sunscreen.
  • ☀️ Afternoon: Reapply SPF & hydrate with face mist.
  • 🌇 Evening: Cleanser → Toner → Serum → Light Moisturizer.
  • 🌙 Night: Double cleanse → Toner → Eye Cream → Night Cream.
  • 💃 Date Night: Primer → Foundation → Lipstick → Highlighter.
  • 🎉 Hangout: BB Cream → Lip Tint → Eyeliner → Compact.
  • 💼 Workday: Light Moisturizer → Compact → Mascara → Lip Balm.
  • 🧘 Self-care Sunday: Exfoliate → Face Mask → Serum → Night Cream.
